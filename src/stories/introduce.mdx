## 시작

### 패키지 설치

이걸 사용하고 싶다면 프로젝트에 다음 패키지가 필수로 설치되어 있어야 한다.

- `react`, `react-dom` v18 이상
- `tailwindcss` v4 이상
- `motion` v12 이상

```bash
# dependencies
pnpm add react@latest react-dom@latest motion@latest

# devDependencies
pnpm add -D tailwindcss@latest
```

Github Packages에 배포되었기 때문에 레지스트리도 바꿔줘야 한다.

그리고 설치하면 된다.

근데 사실 이거 프라이빗 패키지다.

나만 쓸꺼임

```bash
pnpm add @stylelist94/nine-beauty-actress
```
