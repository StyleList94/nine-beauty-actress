# useMounted

컴포넌트가 클라이언트에 완전히 마운트 되는 상태를 추적합니다.

## Return

`mounted`: `true`면 컴포넌트가 마운트 된 상태임을 의미합니다. `boolean`

## Example

<Story of={useMountedStories.Example} />

```tsx
const TargetComponent = () => {
  const isMounted = useMounted();

  return <p>is component mounted?: {isMounted ? 'yes!' : 'no!'}</p>;
};
```
